# Processus ASI

## 1. Support Helpdesk

```
Ticket client → Qualification N1 → Diagnostic → Résolution → Clôture
```

| Étape | Durée | Responsable |
|-------|-------|-------------|
| Ouverture ticket | Immédiat | Client / N1 |
| Qualification | < 1h | N1 |
| Diagnostic | < 2h | N1/N2 |
| Résolution | Variable | N1/N2/N3 |
| Clôture | < 24h | N1 |

## 2. Maintenance Infrastructure

- Supervision continue des serveurs et réseau
- Sauvegardes quotidiennes (rétention 30 jours)
- Mise à jour des correctifs de sécurité
- Gestion des incidents critiques (astreinte)

## 3. Déploiement

```
Demande → Préparation → Test → Déploiement → Validation
```
