# Processus RH

## 🎯 Objectif
> _Cartographier les processus RH_

## 📋 Processus Clés

### 1. Recrutement

```
Besoins → Définition poste → Sourcing → Sélection → Entretien → Embauche → Intégration
```

| Étape | Durée moyenne | Responsable |
|-------|---------------|-------------|
| Identification besoin | 2 jours | Manager |
| Rédaction offre | 3 jours | RH |
| Sourcing | 2-4 sem. | RH |
| Entretiens | 2 sem. | RH + Manager |
| Embauche | 1 sem. | RH |

### 2. Intégration

| Étape | Contenu |
|-------|---------|
| Administrative | Contrat, badges, outils |
| Formation | Welcome pack, formations |
| Tutorat | Buddy assigné |

### 3. Gestion des Congés

- [ ] Saisie demandes
- [ ] Validation manager
- [ ] Suivi solde

### 4. Évaluation

| Période | Type | Méthode |
|---------|------|---------|
| Annuel | Entretien développement | _méthode_ |
| Projets | Feedback continu | _méthode_ |

### 5. Formation

| Processus | Fréquence |
|-----------|-----------|
| Plan formation | Annuel |
| Sessions | Trimestriel |
| Certification | Selon besoin |
