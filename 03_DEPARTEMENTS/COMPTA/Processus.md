# Processus COMPTA

## 1. Facturation Clients

```
Intervention/Commande → Saisie → Facture → Envoi → Suivi → Encaissement
```

| Étape | Délai | Responsable |
|-------|-------|-------------|
| Saisie | < 24h | Comptable |
| Émission facture | < 48h | Comptable |
| Envoi client | Immédiat | Système / Comptable |
| Relance impayé | J+30 | Comptable |

## 2. Comptabilité Fournisseurs

```
Réception facture → Validation → Saisie → Échéance → Paiement
```

## 3. Clôture Comptable

```
Journalisation → Rapprochement → Déclarations fiscales → Clôture
```

## 4. Déclarations Fiscales

- TVA : mensuelle
- IFU / IR : mensuelle
- Bilan annuel : dans les délais légaux
