# NG-STARs 

Bienvenue dans le coffre Obsidian de la **Direction des Systèmes d'Information (DSI)** de NG-STARs.

Ce référentiel centralise toute la connaissance liée à l'audit, la stratégie, les projets et la transformation digitale de l'entreprise.

## 📁 Structure du Coffre

| Dossier         | Contenu                                                               |
| --------------- | --------------------------------------------------------------------- |
| `04_AUDIT/`     | Questionnaires complets, audits IT/financier/processus/RH/stratégique |
| `06_PROJETS/`   | Projets : App Unique, Automatisation, Data, ERP, IA                   |
| `11_TEMPLATES/` | Templates Obsidian pour architecture, audit, cahier des charges, etc. |
| `99_ARCHIVES/`  | Archives et versions antérieures des documents                        |

## 🚀 Projets Clés

- **Projet NG-Fields** — Application terrain pour les interventions
## 🛠️ Technologies

- **Stack actuelle** : Java/Spring Boot, Maven/Gradle, PostgreSQL, Keycloak
- **Cible** : Architecture microservices, cloud, API-first
- **Gestion** : Obsidian pour la connaissance, Git/GitHub pour le versionning

## 📋 Comment Contribuer

1. **Clonez** le dépôt : `git clone https://github.com/moltesse12/NG-STARs.git`
2. **Ouvrez** le coffre dans Obsidian (Ouvrir un dossier comme coffre)
3. **Créez/modifiez** des notes dans les dossiers appropriés
4. **Committez et pushez** : `git add . && git commit -m "message" && git push`
