# Processus RH

## 🎯 Objectif
> _Cartographier les processus Ressources Humaines_

## 📋 Processus Clés

### 1. Recrutement

```
Besoins → Poste → Sourcing → Screening → Entretien(s) → Embauche → Onboarding
```

| Étape | Durée cible | Responsable |
|-------|-------------|-------------|
| Identification besoin | 2 jours | Manager |
| Rédaction offre | 3 jours | RH |
| Sourcing | 2-4 sem. | RH |
| Screening CV | 1 sem. | RH |
| Entretiens | 2 sem. | RH + Manager |
| Décision | 3 jours | Manager |
| Embauche | 1 sem. | RH |

### 2. Onboarding

| Phase | Contenu | Durée |
|-------|---------|-------|
| J-1 | Préparation poste, accès | RH |
| J1 | Welcome,-badge, outils | RH |
| J1-J5 | Formations intro | RH + Manager |
| J5-J30 | Intégration projet | Buddy/Mentor |
| J30 | Point bilan | Manager |

### 3. Gestion des Congés

- [ ] Saisie demande par collaborateur
- [ ] Validation manager
- [ ] Alerte RH si anomalie
- [ ] Mise à jour planning

### 4. Évaluations

| Type | Période | Méthode |
|------|---------|---------|
| Annuel | Janvier | Entretien structuré |
| Probing | Q4 | Auto-évaluation |
| Feedback | Continu | 1:1 hebdo |

### 5. Formation

| Processus | Fréquence | Responsable |
|-----------|-----------|-------------|
| Identification besoins | Q4 | Manager + RH |
| Plan formation | Annuel | RH |
| Sessions | Trimestriel | RH |
| Suivi | Post-formation | RH |

### 6. Sortie Collaborateur

- [ ] Demande départ (résiliation/démission)
- [ ] Entretien exit
- [ ] Calcul solde tout compte
- [ ] Récupération équipements
- [ ] Désactivation comptes
- [ ] Transfert knowledge
